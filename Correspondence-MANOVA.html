1 Introdução CA Corresponden Analysis (AC)

  • AC Redução de dimensão no contexto de variáveis categóricas;

  • AC é uma extensão do PCA;

  • AC é uma abordagem geométrico para visualizar relações entre linhas e colunas de uma tabela de contingência.

  • Exemplo Associação entre duas variáveis. Tabela de dois fatores

2 MANOVA Análise Multivariado de Variância

  • Muitas respostas podem ser analisadas em uma análise variância.

  • Verificar se o peso e altura de pássaros são diferentes simultaneamente para dos tipos de tratamentos.

  • Peso e altura variável dependente.

  • Hipótese é estudar se ambas variáveis juntas são afetadas pelos diferentes tratamentos.

  • Fazer MANOVA

2.1 Suposições

  • As variáveis dependêntes devem seguir uma distribuição normal multivariada;

  • No R usar a função mshapiro.test( ) no pacote mvnormtest;

  • Variâncias homogêneis em todos os grupos;

  • Linearidade em todos as relações dos grupos, cada célula;

  • A conclusão após uma análise MANOVA, é avaliar se o efeito do tratamento é significativo;

  • Compexidade: identificar se o tratamento impactou na

  • variável peso;

  • variável altura;

  • na interação peso, altura;

  • Podemos identificar qual das variáveis dependentes específicas que contribuíram para o efeito global significativo;

  • Podemos usar ANOVA com um fator.
